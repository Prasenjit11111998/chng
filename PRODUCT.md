# Product

## Register

product

## Users
Designers, developers, and privacy-conscious users who regularly process images and need a fast, local alternative to cloud-based upload tools. They value speed, simplicity, keyboard accessibility, and absolute data privacy.

## Product Purpose
A client-side utility toolkit that performs image format conversions and target-size file compression entirely in the browser using WebAssembly. Success is measured by instant load times, zero data leaving the client, and a friction-free "drop-and-download" user experience.

## Brand Personality
- Minimal
- Professional
- Quiet
Voice is utility-focused, avoiding marketing fluff, while maintaining an expert, confidence-inspiring developer aesthetic.

## Anti-references
- Bloated SaaS file converters covered in ads, sign-up prompts, and cookie banners.
- Tools requiring server uploads or imposing file size and speed restrictions.
- Over-designed interfaces that prioritize visual decoration (e.g. glassmorphic glow effects, excessive animations) over utility.

## Design Principles
- **Privacy as a Default**: No network requests should be made for core operations. The user should be able to run the app entirely offline.
- **Zero-setup Frictionless Utility**: Drop files, configure, and download. No wizards, steps, or account creation.
- **Quiet Confidence**: Let typography, alignment, and spacing communicate functionality. Spacing and scale contrast establish hierarchy.
- **No Decoration**: Visual aesthetics should serve functional clarity. Clean borders and monochrome surfaces replace drop shadows and colorful fills.

## Accessibility & Inclusion
- WCAG AA level color contrast for all text.
- Full keyboard navigation support (interactive tabs, focus states, and dropzone trigger accessibility).
- Reduced-motion safe (instant transitions and opacity fades for motion preference).
